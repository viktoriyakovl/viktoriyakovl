# Описание раздела

В данном разделе представлены небольшие      обезличнные примеры моих работ, отражающих

- владение нотациями C4,
- способность самостоятельно по коду описывать верхнеуровневую архитектуру приложений и фич (High-level Arhitecture), 
- умение описывать базы данных и строить ER-диаграммы в PlantUML,

- _BPMN_
